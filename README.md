# TrackPulse

**TrackPulse** — система автоматической фиксации кругов для соревнований радиоуправляемых моделей (RC).

## Описание

TrackPulse автоматизирует судейство RC-соревнований: регистрация участников, управление заездами, автоматическая фиксация кругов через RFID-антенну, трансляция результатов в реальном времени.

## Особенности

- Desktop UI с вкладками для администрирования
- Веб-интерфейс для зрителей в реальном времени
- Поддержка RFID-считывателей EM4095 и RC522
- Многоязычная поддержка (RU/EN)
- Автоматическая фиксация кругов
- WebSocket-обновления в реальном времени
