# TrackPulse

Система фиксации кругов для RC-гонок на базе Arduino + RFID.

## Требования
- Go 1.20+
- Arduino Mega 2560
- Модуль EM4095 или RC522

## Установка
```bash
git clone https://github.com/yourusername/trackpulse.git 
cd trackpulse
make build
```

## Запуск
```bash
./trackpulse
```
